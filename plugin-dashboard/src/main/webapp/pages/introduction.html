Welcome to Hippo Essentials

Hippo Essentials makes it easier to setup a Hippo project. With Essentials, you can browse the Essentials feature library and install extra functionality in your new Hippo project. To get some basic understanding about Essentials and how to use it, check the documentation.

Important Notes

  • Please note that while you are using Essentials, it might be necessary to rebuild your project several times. This is caused by the fact that the plugins need to be integrated into both the Essentials application and your project.
  • Please note that if Essentials encounters a preview HST configuration in your repository, it will be deleted automatically and you will loose your pending changes. Such a preview configuration is the result of changes made in the document template composer that you forgot to publish.

Essentials default settings

Essentials will apply several settings by default. You can set these settings below. It is advised to select add sample content to get started.
