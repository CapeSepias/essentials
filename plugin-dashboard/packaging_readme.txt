#### Prerequisites

* [Node Package Manager](https://npmjs.org/) (NPM)
* [Git](http://git-scm.com/)
* [Subversion](http://subversion.apache.org/)

#### Dependencies

* [Grunt](http://gruntjs.com/) (build tool)
* [Bower](http://bower.io/) (package management)

## Environment setup
#### Install the project dependencies
Run the command below in the project root directory.

    $ npm install && bower install

## Instructions
#### Generate build
To build the application, run `$ grunt build` inside the project directory. You can find the output in the `target/cms/app` directory.
